# 2D Metroidvania Platformer (In Development)

An action-oriented 2D platformer developed in **Unity 2D** using **C#**. This project focuses on building a scalable game architecture, robust physics-based character controllers, and an immersive exploration experience.

## üöÄ Current Technical Highlights
* **State-Driven Character Controller**: Implemented a **Finite State Machine (FSM)** to manage complex player states (Idle, Run, Jump, Wall Slide, Dash, Attack), ensuring smooth transitions and responsive controls.
* **Modular Game Architecture**: Designed decoupled systems for health management, combat hitboxes, and environmental triggers to facilitate code maintainability and scalability.
* **Dynamic Camera System**: Leveraged **Cinemachine** with custom framing transposers and screen shake effects to enhance the "game feel" and player feedback.
